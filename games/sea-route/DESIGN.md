## 核心設計理念（v0.2 確認版）

| 項目 | 內容 |
|------|------|
| 類型 | 放置 Idle + 主動操作 + 探索解鎖 |
| 核心張力 | 放置自動蒐集 vs 手動爆發效率——玩家隨時可以插手提升節奏 |
| 成長感核心 | **紙娃娃系統**：船體外觀隨升級直觀變化（船身→船帆→艙台→裝飾） |
| 解鎖條件 | 蒐集滿該國 N 素材 **+ 打敗 Boss** |
| 離線收益 | **無**——上線才能玩，策略深度優先 |
| 手動互動 | 點擊/滑動手動蒐集可觸發倍率；手動應對 Boss 機制 |

## 4 條升級樹

### ① 航速線 Speed
- 船速 / 燃油效率 / 急速衝刺
- 視覺：引擎變大、煙囪冒煙、船帆鼓脹更明顯

### ② 蒐集線 Collect
- 蒐集範圍 / 自動觸發頻率 / 手動點擊倍率
- 視覺：甲板上多出蒐集旗子、環形光圈變大

### ③ 船體線 Hull
- 防禦強化 / 緊急修復 / 防禦陣形
- 視覺：船身顏色加深、出現裝甲板、船型變厚實

### ④ 特殊能力線 Special
- 每國解鎖 1 個獨特被動技能
- 技能強化
- 終極技能（大招）
- 視覺：船體出現發光效果、特殊標誌

## 10 國 Boss 機制（設計核心）

> Boss 不是純血量關卡，而是**驗收你在該國航行中的升級策略選擇**

| 國家 | Boss 名稱 | 機制設計 | 驗收什麼 |
|------|-----------|---------|---------|
| 🇹🇼 台灣 | 颱風女神 | 逆風減速——必須強化「引擎/船帆」到一定程度才能突破風速 | 航速線 |
| 🇯🇵 日本 | 海賊船長 | 定時搶奪你的素材——要手動「格擋」點擊抵抗 | 手動反應力 |
| 🇰🇷 韓國 | 幽靈鯊魚 | 潛水攻擊船底——要切換「防禦升級分支」才能撐住 | 船體線 |

| 🇻🇳 越南 | 迷霧水鬼 | 視野被遮——靠之前蒐集的「燈籠素材（台灣）」才能驅散 | 跨國素材策略 |

| 🇨🇦 加拿大 | 極光巨獸 | 終極 Boss，融合前面所有機制，需要全船裝備最強 | 全線總驗收 |

## 對話中確認的設計決策

- **蒐集方式**：自動 idle + 手動點擊觸發倍率，兩者並存
- **成長感**：紙娃娃系統，船隻外觀必須隨升級變化
- **解鎖條件**：蒐集 N 素材 + 打敗 Boss（雙條件）
- **離線收益**：無，上線才能玩（策略優先）
- **語言**：繁中 + 英文雙語
